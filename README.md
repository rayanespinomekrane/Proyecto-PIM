# Proyecto_PIM

This is an educational web project that I called Western Informatics, inspired by my favourite classic italian western movies, focused on the stuff we studied during the first year of SMR (Microcomputer Systems and Networks):

- Computer Networks
- Operating Systems
- Computer Assembly and Maintenance

The website was made in visual studio using html, javascript and css.

Discover the website at: https://rayanespinomekrane.github.io/Proyecto-PIM/
